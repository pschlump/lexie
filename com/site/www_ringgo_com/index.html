About RingGo

Ringo is a template engine. There is a command line interface for testing and building static sites. This webpage was build with the command line version of ringo. The templates use a syntax based on Django Templates. -xyzzy- other template systems.

Ringo is implemented in Go and can conventilenly be built into a Go HTTP server.

Ringo uses the Lexie single pass scanner. This gives Ringo the ability to change tokens at runtime.
